# State Class Specification

## State Class Name

`State`

## State Fields

### 1. topic
- **Data Type**: `str`
- **Description**: The report topic provided as input; initial entry point for report generation.

### 2. sections
- **Data Type**: `list[str]`
- **Description**: List of up to 4 section titles generated based on the report topic; used as input list to the `worker` node for section content generation.

### 3. processed_sections
- **Data Type**: `Annotated[list[str], operator.add]`
- **Description**: List used to accumulate the content generated by the worker for each report section; stores the synthesized content for all sections.
