# State Class Specification

## State Class Name

`State`

## State Fields

| Field Name     | Data Type         | Annotation                         | Description                                                        |
|----------------|------------------|------------------------------------|--------------------------------------------------------------------|
| topic          | str               |                                    | The topic for the joke, provided when the graph is invoked.         |
| joke           | str               |                                    | The joke generated based on the topic.                              |
| is_funny       | bool              |                                    | Result of the joke evaluation, set to True or False at random.      |

## Field Details

1. **topic**
    - **Type:** `str`
    - **Description:** The topic that is provided as initial input when the graph is started. It is used as the basis for generating the joke.
    
2. **joke**
    - **Type:** `str`
    - **Description:** The joke generated by the `llm_call_generator` node, based on the provided topic.

3. **is_funny**
    - **Type:** `bool`
    - **Description:** The evaluation of the joke's funniness, determined pseudo-randomly (50% chance of True/False) by `llm_call_evaluator`.
