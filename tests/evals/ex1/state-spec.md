# Exercise_1_State Class Specification

## State Class Name

`Exercise_1_State`

## State Fields

### 1. name
- **Type:** `str`
- **Description:** The name of the person to receive a compliment. This value is provided when invoking the process.

### 2. compliment_text
- **Type:** `str`
- **Description:** The generated compliment for the person, created using their name and saved in state during the `compliment` node.
